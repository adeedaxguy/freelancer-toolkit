## When JPG to PDF is useful

JPG to PDF helps when you need to send several images as one document:

- Receipts
- Scanned forms
- Homework pages
- Client sketches
- ID copies
- Photo evidence
- Application attachments

The PDF format is easier to share because it keeps pages together and opens consistently across devices.

## What to check before sending

Before you upload a PDF to an application portal or email it to a client, open the file and check:

- Page order
- Page orientation
- Whether text is readable
- Total file size
- Whether any private page was included by mistake
