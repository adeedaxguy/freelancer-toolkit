Application portals often ask for a signature image under 20KB. That sounds simple until the file keeps bouncing back because it is too large, too blurry, or the background is not clean enough.

Use the [Resize Signature to 20KB](/tools/resize-signature-to-20kb) tool to create a small JPEG signature file directly in your browser.

## Why signatures are hard to compress

Signatures look simple, but scanned files can be huge because they include:

- A full-page scan instead of only the signature
- Gray paper texture
- Shadows from phone photos
- High-resolution camera metadata
- Extra white space around the actual signature

The fix is to crop tightly around the signature and export only the useful area.

## Best way to prepare the signature

Write the signature on plain white paper using a dark pen. Take a photo in bright, even light. Crop out the rest of the page before reducing file size.

Then open [Resize Signature to 20KB](/tools/resize-signature-to-20kb), upload the image, and download the compressed result. The preset uses a wide signature shape and targets 20KB or less where possible.

## Troubleshooting rejected uploads

If the portal rejects the file, check these details:

| Problem | Fix |
|---|---|
| File is still above 20KB | Lower the quality slider or crop tighter |
| Signature is blurry | Start from the original photo again |
| Background is gray | Retake in brighter light or crop out shadows |
| Portal wants JPG only | Download as JPEG from the tool |
| Signature is too small | Keep the signature wide before compressing |
