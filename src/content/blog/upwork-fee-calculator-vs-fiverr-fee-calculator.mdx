## Step-by-step workflow

1. Choose the platform where the client will actually pay.
2. Enter the project amount and platform fee assumption.
3. Compare net pay, not headline revenue.
4. Add proposal time, revisions, taxes, and payout costs.
5. Adjust the gross quote before you send or publish it.

## FAQ

### Is Upwork cheaper than Fiverr?

It depends on the contract and fee shown. Compare net earnings for the exact project instead of relying on a headline platform rate.

### Which calculator should I use first?

Use the calculator for the platform where the payment will happen. For a bid on Upwork, start with Upwork. For gig packages, start with Fiverr.

### Can I compare both for the same service?

Yes. Run the same target net through both workflows to see what gross price each platform requires.
