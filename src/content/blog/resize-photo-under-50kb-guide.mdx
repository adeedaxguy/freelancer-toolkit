"Photo must be less than 50KB" is one of the most common upload errors on application portals. The problem is that modern phone photos are often 2MB to 8MB before compression.

Use the [Resize Photo Under 50KB](/tools/resize-photo-under-50kb) tool to resize and compress the image in your browser.

## What actually reduces file size

Three things control image file size:

- Pixel dimensions
- File format
- Compression quality

Cropping alone may not be enough. A 3000px-wide image can still be too large even after removing empty space. For upload portals, a smaller JPEG at reasonable quality is usually the safest path.

## Recommended workflow

1. Start with the original image, not a screenshot from chat.
2. Open [Resize Photo Under 50KB](/tools/resize-photo-under-50kb).
3. Upload the photo.
4. Crop or fit it to the preset.
5. Download the compressed JPEG.
6. Check that the file remains clear enough for the application.

If the portal has a different limit, use [Resize Photo Under 20KB](/tools/resize-photo-under-20kb) or [Resize Photo Under 100KB](/tools/resize-photo-under-100kb).

## How to avoid blurry results

Do not repeatedly compress the same file. Each JPEG export loses some detail. If the first result is blurry, go back to the original image and try a different crop or quality setting.

For ID photos, keep the face large enough to identify, but avoid extreme compression. A portal may accept the file size and still reject the photo if the face is unclear.
