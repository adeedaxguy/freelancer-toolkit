## FAQ

### Is 300KB enough for a clear image?

Often yes. It is usually enough for form photos, listing images, CMS uploads, and email attachments, but very detailed images may still need careful cropping.

### Does the tool upload my photo?

No. The image is processed in your browser, so the file stays on your device.

### What if a form asks for 300KB in pixels?

File size and pixel size are different. Use the 300KB preset for file weight, then check whether the form also gives a width and height requirement.

## When 300KB is the right target

Use 300KB when the destination allows a medium-size file but still blocks full-resolution camera photos. It is often a good working target for small product images, headshots, article images, job profiles, local business photos, and CMS uploads where 50KB would make the image visibly rough.

If the form requires a passport or visa photo, follow the official dimension and background rules first. File size is only one part of acceptance. A perfect 300KB export can still fail if the crop, lighting, background, or head position does not match the application rules.

## Quality checklist before uploading

- Check the exported file size on your device before submitting.
- Open the image at normal screen size and confirm the face, text, or product detail is readable.
- Avoid lowering JPEG quality if resizing dimensions already gets close to the target.
- Use fit mode when cropping would remove important information.
- Use a stricter preset only when the final form rejects 300KB.
