## FAQ

### Do I need Excel to calculate a project price?

No. Excel works, but a browser calculator can handle the same inputs faster for simple freelance quotes.

### When is Excel better?

Excel is better for complex recurring models, multi-team estimates, or custom formulas.

### Can I still copy the result into a spreadsheet?

Yes. Use the calculator for the first pass, then copy the result if you keep a pricing archive.
