# SEO Content Briefs

Use this folder for briefs before writing or refreshing posts.

Each brief should include:

- Primary keyword.
- Secondary keywords.
- Search intent.
- Existing page or new page decision.
- Target tool or category hub.
- Ahrefs source and export date.
- SERP competitors to beat.
- Required sections, tables, FAQs, and examples.
- Internal links to add.
- External primary sources if facts can change.
- CTA and measurement plan.

Prefer updating an existing matching tool or blog page before creating a new competing slug.
